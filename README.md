# FileShare — Сервис безопасного обмена файлами
# https://rrrrr23.pythonanywhere.com
Веб-приложение для загрузки и обмена файлами с функцией самоуничтожения по таймеру. Реализован современный интерфейс в стиле Apple, защита данных и автоматическая очистка хранилища.

## Технологии
- **Backend:** Python 3.10, Flask
- **Database:** SQLite + SQLAlchemy ORM
- **Frontend:** HTML5, CSS3 (Glassmorphism), JavaScript (Live Timer)
- **Auth:** Flask-Login (сессии, хеширование паролей)
- **Deployment:** PythonAnywhere Cloud

## Функционал
1. **Авторизация:** Регистрация и вход (защита от неавторизованного доступа).
2. **Загрузка:** Drag-and-Drop интерфейс, валидация расширений.
3. **Таймер жизни:** Выбор времени хранения (от 5 секунд до 7 дней).
4. **Удаление:**
   - Автоматическое удаление файла с диска по истечении времени.
   - Визуальное отображение статуса "Файл удален" без перезагрузки страницы (AJAX-like).
5. **Безопасность:** Хеширование имен файлов (UUID) для защиты от перебора прямых ссылок.

## Установка и запуск
1. Клонировать репозиторий:
   `git clone https://github.com/oaoaoammmm347-dot/flask_fileshare.git`
2. Создать виртуальное окружение:
   `python -m venv venv`
3. Установить зависимости:
   `pip install -r requirements.txt`
4. Запустить:
   `python app.py`
